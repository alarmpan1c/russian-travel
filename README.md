# Проект: Путешествие по России

Проект по третьему спринту от курса веб-разработки от Яндекс.Практикум.

**Технологии, которые были использованы в проекте**
1. HTML
2. CSS
3. Grid Layout
4. Flexbox вёрства
5. Адаптация через медиазапросы
6. Методология БЭМ
